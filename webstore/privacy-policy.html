Briefly 隐私政策

最后更新：2025-08-10

我们收集什么数据

  • 用户主动输入的数据：模型 API Key、模型名称、提示词。
  • 页面处理数据：当前网页的标题、URL、正文文本（用于摘要）。

数据如何使用与存储

  • API Key 与模型配置保存在浏览器的 chrome.storage.sync（或等效存储）中，仅用于调用你选择的模型服务。
  • 网页正文仅在本地提取处理，并通过你的 API Key 直接发送到对应模型服务（例如 DeepSeek 或 DashScope/Qwen）。开发者不会收集、存储或传输这些内容。
  • 摘要历史保存在你的浏览器本地（chrome.storage.local）。你可以随时卸载扩展以移除本地数据。

第三方服务

当你使用扩展进行摘要时，请求会发送到你选择的模型服务，其数据处理遵循第三方的隐私政策。

数据共享

我们不向任何第三方出售、出租或分享你的数据。所有请求均由你的浏览器直接发往模型服务。

安全

扩展采用最小权限原则，仅在用户操作时访问活动标签页。请妥善保管你的 API Key。
